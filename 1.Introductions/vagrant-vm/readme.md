## Get started with Vagrant 

```bash 
vagrant init ubuntu/jammy64
vagrant up 
vagrant status # check status vms
vagrant ssh # login from your terminal 
vagrant ssh-config 
vagrant reload  
vagrant halt 
vagrant destroy -f   
```
